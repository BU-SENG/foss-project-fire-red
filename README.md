# CampusConnect - FOSS Project Fire Red

A student feedback management system with admin dashboard.

## Project Structure

- frontend/admin-dashboard/ - React-based admin interface
- backend/ - Future backend API
- docs/ - Project documentation

## Admin Dashboard

The admin dashboard is located in frontend/admin-dashboard/ and provides:
- Real-time feedback management
- Analytics and reporting
- Admin user management
- Mobile-responsive design
